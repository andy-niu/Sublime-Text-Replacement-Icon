# Sublime Text 2 Replacement Icon #

## Usage ##

    open /Applications/Sublime\ Text\ 2.app/Contents/Resources/

Replace `Sublime Text 2.icns` with the one in this repo.

## Details ##

* Retina Mac ready. All sizes included.
* *Windows icon not tested.*
